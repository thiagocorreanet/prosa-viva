# Prosa Viva v0.1.0

A primeira versão pública do Prosa Viva distribui `$refinar-prosa` como plugin
nativo do Codex e como skill independente.

## Capacidades

- revisa e reescreve prosa em português brasileiro;
- melhora clareza, concisão, ritmo e naturalidade;
- preserva fatos, nomes, números, datas, citações e grau de certeza;
- protege frontmatter, links, tabelas, código, comandos e dados em arquivos;
- atende texto colado, edição de arquivo e uso embutido;
- mantém o texto quando não há melhoria editorial útil.

## Instalação como plugin

```bash
codex plugin marketplace add thiagocorreanet/prosa-viva
codex plugin add prosa-viva@prosa-viva
```

Abra uma conversa nova e invoque `$refinar-prosa`.

## Instalação somente como skill

```bash
npx skills add thiagocorreanet/prosa-viva \
  --skill refinar-prosa \
  --agent codex \
  --global \
  --yes \
  --copy
```

## Limitações

- A série `0.1.x` suporta oficialmente apenas pt-BR.
- O plugin não traduz textos nem verifica fatos.
- O plugin não substitui revisão jurídica, acadêmica ou técnica especializada.
- O plugin não promete resultado em detectores nem afirma autoria humana.
- Esta versão não inclui app, MCP, hooks ou autenticação externa.

Relate problemas e sugestões nas
[issues do projeto](https://github.com/thiagocorreanet/prosa-viva/issues).
